# 3D NOW APP

3D Printing Marketplace.

KEY FEATURES

FOR CUSTOMERS
- Model Marketplace: Browse, Search and Purchase 3D models.
- Custom Design Requests: Upload personal 3D files(supporting .stl, .obj, and .gcode)
with specific material and quality requirements.
- Find Store: Find nearby 3D printing service providers
using real-time location tracking powered by Google Maps API.
- Order Tracking: Manage and track the status of current and past orders.

For ADMIN & DESIGNERS
- Content Management: Manage the marketplace catalog by adding,
editing, or removing 3D models.
- Engagement Tools: Update the "What's New" section
to share latest model and promotions with users.

-------------------------------------------------------------------------------------------------------

Tech Stack
Frontend: Flutter (Dart) for cross-platform mobile development.
- Backend: Firebase:
- Authentication: Secure user login and role management.
- Cloud Firestore: Real-time database for managing marketplace data and user orders.
- Cloud Storage: Secure hosting for 3D model files and preview images.
- APIs: Google Maps API (Nearby Search and Geocoding) for location-based services.
